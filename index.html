Introducción

  • La matemática detrás del diseño y análisis de circuitos digitales es el álgebra de Boole.
  • El álgebra de Boole, como el ágebra convencional, se basa en una serie de postulados y teoremas que permiten reducir una función a su mínima expresión.
  • Las compuertas lógicas son dispositivos electrónicos que permiten implementar funciones Booleanas.
  • Mediante, el álgebra de Boole y las puertas lógicas, se diseñan multiplexores, decodificadores que son útiles en cualquier sistema digital.

Álgebra de Boole.

Definición

El conjunto $\{B, +, \cdot, \stackrel{\overline{}}{} \}$, donde $B$ es el conjunto de elementos o constantes del álgebra, los símbolos $+$ y $\cdot$ son dos operadores binarios y la barra $\stackrel{\overline{}}{}$ es un operador unario es un álgebra Booleana si se cumple con los postulados de Huntington.

Postulados de Huntington

  • Cerradura: Para todos los elementos a y b en el conjunto $B$.
    • $a+b \,$ es un elemento de $B$.
    • $a\cdot b \,$ es un elemento de $B$.
  • Identidad: Para todos los elementos a y b en el conjunto $B$.
    • $a+0 = 0+a = a \,$
    • $b\cdot 1 = 1 \cdot b = b \,$

Postulados de Huntington

  • Conmutatividad: Para todos los elementos a y b en el conjunto $B$.
    • $a+b = b+a \,$
    • $a\cdot b = b\cdot a \,$
  • Distributividad: Para todos los elementos a, b y c en el conjunto $B$.
    • $a \cdot (b+c) = (a\cdot b) + (a \cdot c) \,$
    • $a + (b\cdot c) = (a+b)\cdot (a+c) \,$

Postulados de Huntington

  • Para todo elemento a en el conjunto $B$, existe un elemento $\bar{a}$ en $B$ tal que:
    • $a+\bar{a} = 1 \,$
    • $a\cdot \bar{a} = 0 \,$
  • Existen por lo menos, dos elementos distintos en $B$.

Teorema de Demorgan

  • Un teorema importante en el álgebra Booleana es el teorema de DeMorgan, el cual está dado por:
    • $\overline{a+b} = \overline{a}\cdot \overline{b} \,$
    • $\overline{a\cdot b} = \overline{a} + \overline{b} \,$

Otras propiedades

  • Otras propiedades importantes del álgebra Booleana son:
    • $a+a = a \,$ y $\, a \cdot a = a \,$
    • $a \cdot 0 = 0 \cdot a = 0 \,$ y $\, a +1 = 1+a = 1 \,$
    • $\overline{\overline{(a)}} = a \,$
    • $a + a\cdot b = a \,$ y $\, a \cdot (a+b) = a \,$
    • $a + \bar{a} \cdot b = a+b \,$ y $a \cdot (\bar{a}+b) = a\cdot b \,$

Ejercicios

  • Simplificar la siguiente expresión Booleana: $h(A,B,C) = BC + B\bar{C} + BA$
  • Hallar una versión simplificada de: $f(w,x,y,z) = \bar{w}\bar{x}z + \bar{w}xz + xyz + wxy$
  • Simplifcar la siguiente función: $g(A,B,C,D) = AB\bar{C}+ABC+BCD + \bar{A}CD + A\bar{B}\bar{C}D$
  • Simplifcar la siguiente función: $f(A,B,C,D,E) = A+ \bar{A}B + \bar{A}\bar{B}C + \bar{A}\bar{B}\bar{C}D + \bar{A}\bar{B}\bar{C}\bar{D}E$

Algebra de conmutación

  • Un álgebra de conmutación es un álgebra Booleana en el que el número de elementos en $B \,$ es exactamente $2$.
  • En una álgebra de conmutación, los operadores $+$ y $\cdot$ son llamados OR y AND, mientras que el operador $\stackrel{\overline{}}{}$ se llama NOT.
  • El comportamiento de los operadores AND, OR y NOT puede ser deducido mediante el análisis de los postulados anteriores.

Funciones Booleanas

  • En múltiples ocasiones obtendremos una función Booleana que precisa de una representación más económica en el sentido del número de variables o constantes Booleanas.
  • Para obtener una versión más simple de una función Booleana aplicamos los postulados y teoremas.
  • Una versión más simple de una función tiene por efecto un menor número de compuertas lógicas.

Compuertas lógicas

  • Una compuerta lógica, es un dispositivo electrónico que implementa una función booleana simple.
  • Para funciones lógicas más complejas se interconectan varias compuertas lógicas con el propósito de implementar dicha función.
  • Usualmente, un conjunto completo de funciones Booleanas se utiliza para implementar cualquier función Booleana, por ejemplo, AND, OR y NOT.
